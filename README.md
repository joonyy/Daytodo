### Github 사용 요령
1. main branch 사용하지 않기
2. 본인의 branch에서 작업 후, 본인 branch에서 git add . -> git commit -m" 커밋메시지 " -> git push
3. github 페이지에서 pull request가 생성됨. 누르고 진행해준다!
4. 만약 able to merge가 되지 않는다면, 본인의 branch에서 git pull 을 통해, 최신 업데이트된 main branch 내용을 받아온다.
5. 다시 git push.
